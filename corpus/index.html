알맹이 말뭉치란?
기존의 말뭉치는 문장 또는 어절 형태로 나누어 제공하는 데이터가 대부분이지만 알맹이 말뭉치는 합성어와 신조어에 초점을 두고 AI 학습을 통해 수집한 데이터입니다. 대부분 문장 형태로 제공하는 일반적인 말뭉치와는 차이가 있습니다. 기존의 한국어 형태소 분석기의 명사 추출에서 뚜렷하게 보이는 오류는 아래와 같습니다.

'강남역사거리' => ['강남', '역사', '거리']

합성어에서의 분석 오류


'건강기능식품협회' => ['건강', '기능', '식품', '협회']

고유명사에서의 분석 오류


'갑분싸' => ['갑']

신조어에서의 분석 오류


'아이오아이' => ['아이오', '아이']

신조어에서의 분석 오류

알맹이 말뭉치는 위 오류를 최소화하기 위하여 AI 학습을 통하여 보다 정확한 명사 형태의 데이터를 수집하고 제공하고자 합니다.
알맹이 말뭉치는 크게 4가지 단계의 학습 과정을 통하여 만들어집니다
  1. 데이터 투입 - 기사 제목과 본문 데이터를 투입합니다.
  2. 응집확률 추출 - 투입한 데이터들의 응집확률을 계산합니다.
  3. 비지도 학습 - 기사 제목과 본문 데이터를 투입합니다.
  4. 데이터 검수 - 찾아낸 데이터에서 합성어와 신조어를 수집해 수기로 데이터 검수를 진행합니다.

(하단의 명사들은 전처리된 데이터)

수집한 한국어 말뭉치
알맹이 말뭉치는 자체 뉴스 기사 수집 이외에도 웹상에서 존재하는 한국어 말뭉치 데이터를 수집했습니다. 수집한 말뭉치 데이터를 알맹이 AI에서 학습된 데이터와 대조 및 이를 재학습하여 합성어와 신조어를 수집했습니다.

궁극적으로는 인터넷에 존재하는 접근 가능한 모든 데이터를 이용하여 학습을 진행할 것입니다. 이를 통해 알맹이 말뭉치를 꾸준히 발전시키어 대한민국에 존재하는 모든 신조어 및 합성어를 저장한 데이터베이스를 구축하는 것이 우리의 목표입니다.
